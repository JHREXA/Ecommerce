Aplicación E-Commerce backend en Java y Spring.
Está basada en la arquitectura de microservicios, entre los que se encuentran:
- Un microservicio encargado del carro de la compra --> Cart
- Un microservicio encargado de cada producto --> Product
- Un microservicio encargado de cada venta --> Sale
- Un microservicio encargado del API Gateway, que funciona como punto de acceso principal a la aplicación.
- Un microservicio para Eureka Server, que servirá para comunicar los microservicios entre sí.
