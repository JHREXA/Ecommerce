# E-Commerce Backend Application

Backend e-commerce application developed with Java, Spring Boot and a microservices architecture.

This project was created as a personal learning project to practice backend development, REST APIs, service separation and microservices communication.

## Project Status

Personal project in progress.

The goal is to improve my skills in Java, Spring Boot, Spring Cloud and microservices architecture.

## Architecture

The application is composed of several backend services:

- Product Service: manages products
- Cart Service: manages shopping carts
- Sale Service: manages sales
- API Gateway: central entry point for the application
- Eureka Server: service discovery

## Tech Stack

- Java
- Spring Boot
- Spring Cloud
- Eureka Server
- API Gateway
- Maven
- Docker
- REST APIs
- Git

## Main Features

- Product management
- Cart management
- Sale management
- REST API communication
- Service discovery with Eureka
- Centralized access through API Gateway

## Learning Goals

This project helped me practice:

- Building REST APIs with Spring Boot
- Structuring a backend application with multiple services
- Understanding API Gateway and service discovery
- Separating responsibilities between services
- Working with Docker-based environments
